# Sarateal

Sarateal is a lightweight Kenya-wide farmer market access radar.

It helps farmers, cooperatives, aggregators, buyers, NGOs, counties, and food distributors connect supply to real demand through market opportunities, tenders, prices, and simple risk signals.

## MVP focus

- Farmer supply registration
- Buyer demand registration
- Tender and opportunity listing
- Simple product-location-volume matching
- Basic shortage and opportunity scoring
- Kenya-wide county structure
- Streamlit dashboard first, API backend underneath

## Initial commodities

- Maize
- Beans
- Potatoes
- Tomatoes
- Onions

## Stack

- Python
- FastAPI
- PostgreSQL/PostGIS or DuckDB
- Streamlit
- CSV / GeoParquet
- Scheduled Python jobs

## Development rule

Build one file at a time.

Tests will be added after the first MVP structure is in place.
